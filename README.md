# Climate Action Co-Benefits Atlas

This Streamlit application visualizes the co-benefits of climate action for the "The Data Lab Data Visualisation Competition 2025".

## Project Structure
- `app.py`: Main application entry point.
- `src/data.py`: Data loading and preprocessing logic.
- `src/visualizations.py`: Plotly visualization functions.
- `requirements.txt`: Python dependencies.
- `Level_3.xlsx`: The dataset (Must be present in this directory).

## How to Run

1. **Install Dependencies** (if not already done):
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the App**:
   ```bash
   streamlit run app.py
   ```

## Features
- **Projected Benefits Timeline**: View how benefits grow from 2025 to 2050.
- **Benefit Composition**: Breakdown of benefits for 2050.
- **Top Area Comparison**: Compare your area with others.
- **Premium UI**: Dark mode and custom styling.
